# 🥃 SYSO - Select Your Spirit Online

> Java Swing 기반 위스키 전문 쇼핑몰 데스크탑 애플리케이션

<br>

## 📌 프로젝트 소개

**SYSO(Select Your Spirit Online)** 는 위스키 애호가를 위한 온라인 주류 쇼핑몰 서비스입니다.
Java Swing으로 구현한 데스크탑 앱으로, 싱글몰트·블렌디드·버번·일본·뉴월드 등 다양한 카테고리의 위스키 40여 종을 탐색하고
장바구니 담기 → 결제 → 주문 내역 확인까지의 전체 쇼핑 플로우를 경험할 수 있습니다.

> **타겟층** : 만 19세 이상 성인, 위스키 애호가

## ✨ 주요 기능

### 01 회원가입
- ID, PW, 이름, 연락처, 이메일, 생년월일 입력
- **만 19세 미만 가입 제한**
- 비밀번호: 영문 대/소문자·숫자·특수문자 포함 9자 이상
- 이메일 형식, 전화번호(010 시작 11자리) 유효성 검사
- 가입 완료 시 다이얼로그 출력 후 로그인 페이지로 이동

### 02 로그인
- ID·비밀번호 입력을 통한 인증
- 로그인 완료 시 메인 페이지로 이동

### 03 메인 페이지
- 위스키 카테고리별 탭 메뉴 (전체보기 / 싱글몰트 / 블렌디드 / 버번 / 일본 / 뉴월드)
- 재고 수량에 따른 라벨 색상 분류 (여유 / 소량 / 재고 없음 / 품절)
- 상단 검색창, 마이 페이지·장바구니 아이콘 제공

### 04 상품 검색
- 키워드 입력 기반 상품명 검색 (대소문자·공백 무시)
- 검색 결과 없을 시 안내 메시지 표시

### 05 상품 상세 페이지
- 상품 이미지, 상품명, 카테고리, 가격, 재고 수량, 상세 설명 제공
- 장바구니 담기 / 바로 결제하기 버튼
- 품절 상품은 구매 불가 처리

### 06 장바구니
- 상품 수량 변경(+/-) 및 선택 삭제 / 전체 삭제
- 총 금액 자동 계산 (배송비 무료)
- 결제 수단 선택 (신용카드 / 간편 결제 / 계좌이체)
- 결제 완료 시 주문 확인 다이얼로그 출력 (주문번호·시간·결제수단·상품 목록·총 금액)

### 07 마이 페이지
- 회원 정보 조회 (이름, 아이디, 생년월일, 전화번호, 이메일)
- 이메일 수정 기능
- 주문 내역 날짜 내림차순 조회

## 🚀 실행 방법

**요구 사항:** Java 11 이상

1. Eclipse에서 프로젝트 import
   ```
   File → Import → Existing Projects into Workspace
   ```
2. `src/main/Main.java` 실행 (`Run As → Java Application`)

> ⚠️ `Img/`, `img_info/`, `ui_img/` 폴더가 프로젝트 루트에 있어야 이미지가 정상 표시됩니다.
